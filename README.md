## Overview

Obsidian v1.4 is a modern desktop workspace focused on productivity, automation, AI-inspired workflows, and smart organization.

This project combines a clean interface with powerful customization and modern desktop technologies to create a fast and scalable workspace experience.

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Obsidian-v1.4.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build the application:

```bash
npm run build
```
